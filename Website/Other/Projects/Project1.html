Welcome to the GitHub Setup and Usage Guide

This guide will walk you through setting up a GitHub account, understanding basic Git operations, and using GitHub Desktop.

Initial Setup

How to create and configure your GitHub account:

  • Visit github.com and sign up for a new account.
  • Follow the prompts to complete your profile setup.
  • Verify your email address to fully activate your account.

Basic Git Commands

Essential commands to manage your projects on GitHub:

  • git clone [url] - Download a repository from GitHub to your local machine.
  • git add [file] - Add files to your next commit.
  • git commit -m "message" - Commit your changes with a descriptive message.
  • git push - Upload your commit to GitHub.
  • git pull - Update your local repository with the latest changes from GitHub.

Using GitHub Desktop

Simplify your Git workflow with GitHub Desktop:

  • Download and install GitHub Desktop from desktop.github.com.
  • Connect GitHub Desktop to your GitHub account to manage your repositories.
  • Use the graphical interface to stage changes, commit them, and push to GitHub without using command line commands.
