Welcome to C# Project 8 - Word Counter

This project will guide you through creating a simple console application in C# that counts the number of words in a given string. It's designed for beginners to practice string manipulation and basic algorithms in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax and data types.
  • String manipulation.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n WordCounter
  4. Navigate to the project directory: 
    cd WordCounter

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace WordCounter
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Word Counter Application");
            Console.WriteLine("Enter a sentence:");
            string input = Console.ReadLine();

            int wordCount = CountWords(input);

            Console.WriteLine($"The number of words in the sentence is: {wordCount}");
        }

        static int CountWords(string input)
        {
            if (string.IsNullOrWhiteSpace(input))
            {
                return 0;
            }

            string[] words = input.Split(' ', StringSplitOptions.RemoveEmptyEntries);
            return words.Length;
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by entering different sentences and checking the word count.

Congratulations! You've created and run your first C# word counter application. Now you're ready to tackle more complex projects.
