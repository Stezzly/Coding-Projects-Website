Welcome to C# Project 10 - Simple Quiz Game

This project will guide you through creating a simple console application in C# that quizzes the user with multiple-choice questions and provides a score at the end. It's designed for beginners to practice arrays or lists, user input, and control flow in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax and data types.
  • Control flow and arrays or lists.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n SimpleQuizGame
  4. Navigate to the project directory: 
    cd SimpleQuizGame

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace SimpleQuizGame
{
    class Program
    {
        static void Main(string[] args)
        {
            string[] questions = {
                "What is the capital of France?",
                "What is 2 + 2?",
                "What is the chemical symbol for water?"
            };

            string[][] options = {
                new string[] {"A. Paris", "B. London", "C. Rome", "D. Berlin"},
                new string[] {"A. 3", "B. 4", "C. 5", "D. 6"},
                new string[] {"A. H2O", "B. CO2", "C. O2", "D. N2"}
            };

            char[] answers = {'A', 'B', 'A'};
            int score = 0;

            for (int i = 0; i < questions.Length; i++)
            {
                Console.WriteLine(questions[i]);
                foreach (string option in options[i])
                {
                    Console.WriteLine(option);
                }
                Console.WriteLine("Enter your answer (A, B, C, or D):");
                char userAnswer = Console.ReadLine().ToUpper()[0];

                if (userAnswer == answers[i])
                {
                    score++;
                    Console.WriteLine("Correct!");
                }
                else
                {
                    Console.WriteLine("Wrong!");
                }
                Console.WriteLine();
            }

            Console.WriteLine($"Your total score is: {score}/{questions.Length}");
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by answering the quiz questions and checking your score.

Congratulations! You've created and run your first C# simple quiz game application. Now you're ready to tackle more complex projects.
