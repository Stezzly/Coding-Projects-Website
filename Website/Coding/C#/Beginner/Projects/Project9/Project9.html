Welcome to C# Project 9 - Unit Converter

This project will guide you through creating a simple console application in C# that converts units (e.g., distance, weight, volume) from one system to another. It's designed for beginners to practice mathematical operations and user input in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax and data types.
  • Mathematical operations and user input.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n UnitConverter
  4. Navigate to the project directory: 
    cd UnitConverter

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace UnitConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Unit Converter Application");
            Console.WriteLine("Enter the value to convert:");
            double value = Convert.ToDouble(Console.ReadLine());

            Console.WriteLine("Enter the source unit (m for meters, kg for kilograms, l for liters):");
            string sourceUnit = Console.ReadLine().ToLower();

            Console.WriteLine("Enter the target unit (ft for feet, lb for pounds, gal for gallons):");
            string targetUnit = Console.ReadLine().ToLower();

            double convertedValue = ConvertUnits(value, sourceUnit, targetUnit);

            if (convertedValue != -1)
            {
                Console.WriteLine($"Converted value: {convertedValue} {targetUnit}");
            }
            else
            {
                Console.WriteLine("Invalid unit conversion.");
            }
        }

        static double ConvertUnits(double value, string sourceUnit, string targetUnit)
        {
            switch (sourceUnit)
            {
                case "m":
                    switch (targetUnit)
                    {
                        case "ft":
                            return value * 3.28084;
                    }
                    break;
                case "kg":
                    switch (targetUnit)
                    {
                        case "lb":
                            return value * 2.20462;
                    }
                    break;
                case "l":
                    switch (targetUnit)
                    {
                        case "gal":
                            return value * 0.264172;
                    }
                    break;
            }
            return -1;
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by converting various units and checking the results.

Congratulations! You've created and run your first C# unit converter application. Now you're ready to tackle more complex projects.
