Welcome to C# Project 4 - Temperature Converter

This project will guide you through creating a simple console application in C# that converts temperatures between Celsius, Fahrenheit, and Kelvin. It's designed for beginners to practice mathematical operations and conditionals in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax and data types.
  • Mathematical operations and conditionals.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n TemperatureConverter
  4. Navigate to the project directory: 
    cd TemperatureConverter

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace TemperatureConverter
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Temperature Converter");
            Console.WriteLine("Enter the temperature value:");
            double temp = Convert.ToDouble(Console.ReadLine());

            Console.WriteLine("Enter the unit of the temperature (C for Celsius, F for Fahrenheit, K for Kelvin):");
            string unit = Console.ReadLine().ToUpper();

            double tempInCelsius;

            switch (unit)
            {
                case "C":
                    tempInCelsius = temp;
                    break;
                case "F":
                    tempInCelsius = (temp - 32) * 5 / 9;
                    break;
                case "K":
                    tempInCelsius = temp - 273.15;
                    break;
                default:
                    Console.WriteLine("Invalid unit.");
                    return;
            }

            double tempInFahrenheit = (tempInCelsius * 9 / 5) + 32;
            double tempInKelvin = tempInCelsius + 273.15;

            Console.WriteLine($"Temperature in Celsius: {tempInCelsius} C");
            Console.WriteLine($"Temperature in Fahrenheit: {tempInFahrenheit} F");
            Console.WriteLine($"Temperature in Kelvin: {tempInKelvin} K");
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by converting various temperatures and checking if the results are correct.

Congratulations! You've created and run your first C# temperature converter application. Now you're ready to tackle more complex projects.
