Welcome to C# Project 6 - Basic ATM Simulation

This project will guide you through creating a simple console application in C# that simulates basic ATM operations like checking balance, depositing money, and withdrawing money. It's designed for beginners to practice working with control flow and basic data management in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax and data types.
  • Control flow (if statements, loops).

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n BasicATMSimulation
  4. Navigate to the project directory: 
    cd BasicATMSimulation

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace BasicATMSimulation
{
    class Program
    {
        static void Main(string[] args)
        {
            double balance = 1000.00;
            string command = "";

            Console.WriteLine("Basic ATM Simulation");
            Console.WriteLine("Commands: balance, deposit [amount], withdraw [amount], exit");

            while (command != "exit")
            {
                Console.WriteLine("Enter a command:");
                command = Console.ReadLine().Trim();
                
                if (command == "balance")
                {
                    Console.WriteLine($"Your current balance is: ${balance}");
                }
                else if (command.StartsWith("deposit "))
                {
                    if (double.TryParse(command.Substring(8), out double amount) && amount > 0)
                    {
                        balance += amount;
                        Console.WriteLine($"Deposited: ${amount}. New balance: ${balance}");
                    }
                    else
                    {
                        Console.WriteLine("Invalid deposit amount.");
                    }
                }
                else if (command.StartsWith("withdraw "))
                {
                    if (double.TryParse(command.Substring(9), out double amount) && amount > 0)
                    {
                        if (amount <= balance)
                        {
                            balance -= amount;
                            Console.WriteLine($"Withdrew: ${amount}. New balance: ${balance}");
                        }
                        else
                        {
                            Console.WriteLine("Insufficient funds.");
                        }
                    }
                    else
                    {
                        Console.WriteLine("Invalid withdrawal amount.");
                    }
                }
                else if (command != "exit")
                {
                    Console.WriteLine("Unknown command.");
                }
            }

            Console.WriteLine("Exiting ATM Simulation.");
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by checking the balance, depositing, and withdrawing money to ensure everything works as expected.

Congratulations! You've created and run your first C# basic ATM simulation application. Now you're ready to tackle more complex projects.
