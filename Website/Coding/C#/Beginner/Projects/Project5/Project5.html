Welcome to C# Project 5 - Simple To-Do List

This project will guide you through creating a simple console application in C# that allows users to add, view, and remove tasks from a to-do list. It's designed for beginners to practice working with lists and basic data management in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax, lists, and control flow.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n SimpleToDoList
  4. Navigate to the project directory: 
    cd SimpleToDoList

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;
using System.Collections.Generic;

namespace SimpleToDoList
{
    class Program
    {
        static void Main(string[] args)
        {
            List toDoList = new List();
            string command = "";

            Console.WriteLine("Simple To-Do List Application");
            Console.WriteLine("Commands: add [task], remove [task number], list, exit");

            while (command != "exit")
            {
                Console.WriteLine("Enter a command:");
                command = Console.ReadLine().Trim();
                
                if (command.StartsWith("add "))
                {
                    string task = command.Substring(4);
                    toDoList.Add(task);
                    Console.WriteLine($"Added task: {task}");
                }
                else if (command.StartsWith("remove "))
                {
                    if (int.TryParse(command.Substring(7), out int taskNumber) && taskNumber > 0 && taskNumber <= toDoList.Count)
                    {
                        string removedTask = toDoList[taskNumber - 1];
                        toDoList.RemoveAt(taskNumber - 1);
                        Console.WriteLine($"Removed task: {removedTask}");
                    }
                    else
                    {
                        Console.WriteLine("Invalid task number.");
                    }
                }
                else if (command == "list")
                {
                    Console.WriteLine("To-Do List:");
                    for (int i = 0; i < toDoList.Count; i++)
                    {
                        Console.WriteLine($"{i + 1}. {toDoList[i]}");
                    }
                }
                else if (command != "exit")
                {
                    Console.WriteLine("Unknown command.");
                }
            }

            Console.WriteLine("Exiting To-Do List Application.");
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by adding, listing, and removing tasks to ensure everything works as expected.

Congratulations! You've created and run your first C# simple to-do list application. Now you're ready to tackle more complex projects.
