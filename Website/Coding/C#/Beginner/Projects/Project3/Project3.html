Welcome to C# Project 3 - Number Guessing Game

This project will guide you through creating a simple console application in C# that allows the user to guess a randomly generated number. It's designed for beginners to practice loops, conditionals, and user input in C#.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or Visual Studio - These are popular Integrated Development Environments (IDEs) for C# development.
  • .NET SDK - Required to compile and run C# programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.
  • Basic C# syntax, loops, and conditionals.

Project Steps:

  1. Download and install Visual Studio or Visual Studio Code.
  2. Download and install the .NET SDK.

  1. Open your chosen IDE (Visual Studio or Visual Studio Code).
  2. Open a terminal in your IDE.
  3. Type the following command to create a new console application: 
    dotnet new console -n NumberGuessingGame
  4. Navigate to the project directory: 
    cd NumberGuessingGame

  1. Open the `Program.cs` file in your IDE.
  2. Replace the existing code with the following: 
    
using System;

namespace NumberGuessingGame
{
    class Program
    {
        static void Main(string[] args)
        {
            Random random = new Random();
            int numberToGuess = random.Next(1, 101);
            int userGuess = 0;
            int guessCount = 0;

            Console.WriteLine("Welcome to the Number Guessing Game!");
            Console.WriteLine("I'm thinking of a number between 1 and 100. Can you guess what it is?");

            while (userGuess != numberToGuess)
            {
                Console.WriteLine("Enter your guess:");
                userGuess = Convert.ToInt32(Console.ReadLine());
                guessCount++;

                if (userGuess > numberToGuess)
                {
                    Console.WriteLine("Too high! Try again.");
                }
                else if (userGuess < numberToGuess)
                {
                    Console.WriteLine("Too low! Try again.");
                }
                else
                {
                    Console.WriteLine($"Congratulations! You've guessed the number in {guessCount} attempts.");
                }
            }
        }
    }
}
  3. Save the file.

  1. In the terminal, type the following command to run your application: 
    dotnet run
  2. Test the application by guessing the number and checking if the responses are correct.

Congratulations! You've created and run your first C# number guessing game application. Now you're ready to tackle more complex projects.
