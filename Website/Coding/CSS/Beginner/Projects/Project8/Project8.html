Welcome to CSS Project 8 - Animated CSS Button

This project will guide you through creating an animated button using HTML and CSS. It's designed for beginners with basic knowledge of HTML and CSS.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Any code editor (e.g., Visual Studio Code, Sublime Text).
  • A web browser (e.g., Chrome, Firefox).

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of HTML and CSS.

Project Steps:

  1. Download and install Visual Studio Code or your preferred code editor.
  2. Open your code editor and create a new folder for your project.
  3. Inside the folder, create a new file named index.html and another file named styles.css.

  1. Open index.html and add the following basic HTML structure: 
    



    
    
    Animated CSS Button

  1. Open styles.css and add the following CSS to style your animated button: 
    
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.button-container {
    text-align: center;
}

.animated-button {
    background-color: #28a745;
    color: #fff;
    padding: 1em 2em;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 1em;
    transition: transform 0.3s ease, background-color 0.3s ease;
}

.animated-button:hover {
    background-color: #218838;
    transform: scale(1.1);
}
  2. Save the file and open index.html in your web browser to see your animated CSS button.

Congratulations! You've created an animated CSS button using HTML and CSS. Now you're ready to tackle more complex projects.
