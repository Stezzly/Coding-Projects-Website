Welcome to CSS Project 5 - Interactive Product Card

This project will guide you through creating an interactive product card using HTML and CSS. It's designed for beginners with basic knowledge of HTML and CSS.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Any code editor (e.g., Visual Studio Code, Sublime Text).
  • A web browser (e.g., Chrome, Firefox).

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of HTML and CSS.

Project Steps:

  1. Download and install Visual Studio Code or your preferred code editor.
  2. Open your code editor and create a new folder for your project.
  3. Inside the folder, create a new file named index.html and another file named styles.css.

  1. Open index.html and add the following basic HTML structure: 
    



    
    
    Interactive Product Card
    


    
    
        
        
    Product Title
    
        
    This is a short description of the product.

  1. Open styles.css and add the following CSS to style your product card: 
    
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.product-card {
    background-color: #fff;
    border-radius: 10px;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
    overflow: hidden;
    width: 300px;
    text-align: center;
    transition: transform 0.3s ease;
}

.product-card:hover {
    transform: translateY(-10px);
}

.product-image {
    width: 100%;
    height: auto;
}

.product-title {
    font-size: 1.5em;
    margin: 1em 0;
}

.product-description {
    font-size: 1em;
    margin: 0 1em 1em;
}

.buy-button {
    background-color: #28a745;
    color: #fff;
    border: none;
    padding: 0.75em 1.5em;
    margin-bottom: 1em;
    cursor: pointer;
    transition: background-color 0.3s ease;
}

.buy-button:hover {
    background-color: #218838;
}
  2. Save the file and open index.html in your web browser to see your styled interactive product card.

Congratulations! You've created an interactive product card using HTML and CSS. Now you're ready to tackle more complex projects.
