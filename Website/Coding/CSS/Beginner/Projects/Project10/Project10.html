Welcome to CSS Project 10 - CSS Animation with Keyframes

This project will guide you through creating an animation using CSS keyframes. It's designed for beginners with basic knowledge of HTML and CSS.

  • A keyboard and mouse for typing and navigation.

  • Basic understanding of HTML and CSS.

Project Steps:

  1. Download and install Visual Studio Code or your preferred code editor.
  2. Open your code editor and create a new folder for your project.
  3. Inside the folder, create a new file named index.html and another file named styles.css.

  1. Open index.html and add the following basic HTML structure: 
    



    
    
    CSS Animation with Keyframes

  1. Open styles.css and add the following CSS to style your animation: 
    
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.animation-container {
    text-align: center;
}

.animated-box {
    width: 100px;
    height: 100px;
    background-color: #28a745;
    animation: moveBox 2s infinite alternate;
}

@keyframes moveBox {
    0% {
        transform: translateX(0);
    }
    100% {
        transform: translateX(200px);
    }
}
  2. Save the file and open index.html in your web browser to see your animated CSS box.

Congratulations! You've created an animation using CSS keyframes. Now you're ready to tackle more complex projects.
