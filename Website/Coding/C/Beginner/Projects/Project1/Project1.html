Welcome to C Programming - Calculating the Sum of Two Integers

This guide will help you write a simple C program that calculates the sum of two integers. It's designed for those who are just starting with C programming.

Setup Requirements

Before you begin, make sure you have the following:

  • A text editor (Notepad++, Sublime Text, or any IDE like Code::Blocks)
  • A C compiler (GCC recommended)

Difficulty Rating

Very Easy Difficulty



Project Time!!!

0%

Project Steps:

Create a new text file and enter the following C code:

#include <stdio.h>

int main() {
    int num1, num2, sum;

    printf("Enter two integers: ");
    scanf("%d %d", &num1, &num2);

    sum = num1 + num2;

    printf("Sum: %d", sum);

    return 0;
}

This program takes two integers as input and prints their sum.

Use your C compiler to compile the code. For GCC, you can use the following command in the terminal:

gcc -o sumProgram yourProgram.c

Replace "yourProgram.c" with the name of your file.

After compiling the code, run the executable:

./sumProgram

Follow the on-screen prompts to enter two integers and get their sum displayed.

Congratulations! You have written your first C program to calculate the sum of two integers.
