Welcome to C++ Project 1 - Hello World Program

This project will guide you through creating a simple console application in C++ that prints "Hello, World!" to the screen. It's designed for beginners with no prior experience in programming.

Hardware Setup

Ensure you have the following hardware components:

  • A computer with internet access.
  • A keyboard and mouse for typing and navigation.

Software Requirements

Here are the software tools you'll need for this project:

  • Visual Studio Code or CLion - These are popular Integrated Development Environments (IDEs) for C++ development.
  • GCC or Visual Studio - Required to compile and run C++ programs.

Knowledge Prerequisites

To successfully complete this project, you should be familiar with the following concepts:

  • Basic understanding of computer operation.

Project Steps:

  1. Download and install Visual Studio Code or CLion.
  2. Ensure you have a C++ compiler installed. For example, you can install GCC or use the built-in compiler in Visual Studio.

  1. Open your chosen IDE (Visual Studio Code or CLion).
  2. Create a new file named hello_world.cpp.

  1. Open the hello_world.cpp file in your IDE.
  2. Type the following code: 
    
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
  3. Save the file.

  1. Open the terminal in your IDE and navigate to the directory containing hello_world.cpp.
  2. Type the following command to compile your application: 
    g++ hello_world.cpp -o hello_world
  3. Type the following command to run your application: 
    ./hello_world
  4. You should see the output: Hello, World!

Congratulations! You've created and run your first C++ console application. Now you're ready to tackle more complex projects.
