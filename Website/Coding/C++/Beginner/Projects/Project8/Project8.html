Project Steps:

  1. Download and install Visual Studio Code or CLion.
  2. Ensure you have a C++ compiler installed. For example, you can install GCC or use the built-in compiler in Visual Studio.

  1. Open your chosen IDE (Visual Studio Code or CLion).
  2. Create a new file named file_io_operations.cpp.

  1. Open the file_io_operations.cpp file in your IDE.
  2. Type the following code: 
    
#include <iostream>
#include <fstream>
#include <string>

using namespace std;

void writeFile(const string &filename) {
    ofstream outFile(filename);
    if (outFile.is_open()) {
        outFile << "Hello, this is a sample text written to the file." << endl;
        outFile.close();
        cout << "File written successfully." << endl;
    } else {
        cout << "Unable to open file for writing." << endl;
    }
}

void readFile(const string &filename) {
    ifstream inFile(filename);
    if (inFile.is_open()) {
        string line;
        while (getline(inFile, line)) {
            cout << line << endl;
        }
        inFile.close();
    } else {
        cout << "Unable to open file for reading." << endl;
    }
}

int main() {
    string filename = "sample.txt";
    writeFile(filename);
    readFile(filename);
    return 0;
}
  3. Save the file.

  1. Open the terminal in your IDE and navigate to the directory containing file_io_operations.cpp.
  2. Type the following command to compile your application: 
    g++ file_io_operations.cpp -o file_io_operations
  3. Type the following command to run your application: 
    ./file_io_operations
  4. Check the output to see the text written to the file and then read from the file.

Congratulations! You've created and run a file I/O operations program in C++. Now you're ready to tackle more complex projects.
