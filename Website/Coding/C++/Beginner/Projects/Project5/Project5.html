Project Steps:

  1. Download and install Visual Studio Code or CLion.
  2. Ensure you have a C++ compiler installed. For example, you can install GCC or use the built-in compiler in Visual Studio.

  1. Open your chosen IDE (Visual Studio Code or CLion).
  2. Create a new file named todo_list.cpp.

  1. Open the todo_list.cpp file in your IDE.
  2. Type the following code: 
    
#include <iostream>
#include <vector>
#include <string>

using namespace std;

void addTask(vector<string> &tasks) {
    cout << "Enter a new task: ";
    string task;
    cin.ignore();
    getline(cin, task);
    tasks.push_back(task);
}

void removeTask(vector<string> &tasks) {
    cout << "Enter the number of the task to remove: ";
    int taskNumber;
    cin >> taskNumber;
    if (taskNumber > 0 && taskNumber <= tasks.size()) {
        tasks.erase(tasks.begin() + taskNumber - 1);
    } else {
        cout << "Invalid task number" << endl;
    }
}

void displayTasks(const vector<string> &tasks) {
    cout << "Your tasks:" << endl;
    for (int i = 0; i < tasks.size(); ++i) {
        cout << i + 1 << ". " << tasks[i] << endl;
    }
}

int main() {
    vector<string> tasks;
    int choice;

    do {
        cout << "1. Add Task" << endl;
        cout << "2. Remove Task" << endl;
        cout << "3. Display Tasks" << endl;
        cout << "4. Exit" << endl;
        cout << "Enter your choice: ";
        cin >> choice;

        switch (choice) {
            case 1:
                addTask(tasks);
                break;
            case 2:
                removeTask(tasks);
                break;
            case 3:
                displayTasks(tasks);
                break;
            case 4:
                cout << "Goodbye!" << endl;
                break;
            default:
                cout << "Invalid choice" << endl;
        }
    } while (choice != 4);

    return 0;
}
  3. Save the file.

  1. Open the terminal in your IDE and navigate to the directory containing todo_list.cpp.
  2. Type the following command to compile your application: 
    g++ todo_list.cpp -o todo_list
  3. Type the following command to run your application: 
    ./todo_list
  4. Use the menu to add, remove, and display tasks.

Congratulations! You've created and run a basic to-do list in C++. Now you're ready to tackle more complex projects.
