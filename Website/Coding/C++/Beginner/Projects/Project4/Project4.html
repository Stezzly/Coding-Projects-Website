Project Steps:

  1. Download and install Visual Studio Code or CLion.
  2. Ensure you have a C++ compiler installed. For example, you can install GCC or use the built-in compiler in Visual Studio.

  1. Open your chosen IDE (Visual Studio Code or CLion).
  2. Create a new file named temperature_converter.cpp.

  1. Open the temperature_converter.cpp file in your IDE.
  2. Type the following code: 
    
#include <iostream>
using namespace std;

void celsiusToFahrenheit(float celsius) {
    float fahrenheit = (celsius * 9 / 5) + 32;
    cout << celsius << "¬∞C is " << fahrenheit << "¬∞F" << endl;
}

void fahrenheitToCelsius(float fahrenheit) {
    float celsius = (fahrenheit - 32) * 5 / 9;
    cout << fahrenheit << "¬∞F is " << celsius << "¬∞C" << endl;
}

int main() {
    int choice;
    float temperature;

    cout << "Temperature Converter" << endl;
    cout << "1. Celsius to Fahrenheit" << endl;
    cout << "2. Fahrenheit to Celsius" << endl;
    cout << "Enter your choice (1 or 2): ";
    cin >> choice;

    if (choice == 1) {
        cout << "Enter temperature in Celsius: ";
        cin >> temperature;
        celsiusToFahrenheit(temperature);
    } else if (choice == 2) {
        cout << "Enter temperature in Fahrenheit: ";
        cin >> temperature;
        fahrenheitToCelsius(temperature);
    } else {
        cout << "Invalid choice" << endl;
    }

    return 0;
}
  3. Save the file.

  1. Open the terminal in your IDE and navigate to the directory containing temperature_converter.cpp.
  2. Type the following command to compile your application: 
    g++ temperature_converter.cpp -o temperature_converter
  3. Type the following command to run your application: 
    ./temperature_converter
  4. Choose the conversion option and enter the temperature to see the result.
