Project Steps:

  1. Download and install Visual Studio Code or CLion.
  2. Ensure you have a C++ compiler installed. For example, you can install GCC or use the built-in compiler in Visual Studio.

  1. Open your chosen IDE (Visual Studio Code or CLion).
  2. Create a new file named factorial_calculator.cpp.

  1. Open the factorial_calculator.cpp file in your IDE.
  2. Type the following code: 
    
#include <iostream>

using namespace std;

// Iterative method to calculate factorial
unsigned long long factorialIterative(int n) {
    unsigned long long result = 1;
    for (int i = 1; i <= n; ++i) {
        result *= i;
    }
    return result;
}

// Recursive method to calculate factorial
unsigned long long factorialRecursive(int n) {
    if (n <= 1)
        return 1;
    else
        return n * factorialRecursive(n - 1);
}

int main() {
    int number;

    cout << "Enter a positive integer: ";
    cin >> number;

    if (number < 0) {
        cout << "Factorial is not defined for negative numbers." << endl;
    } else {
        cout << "Iterative: " << number << "! = " << factorialIterative(number) << endl;
        cout << "Recursive: " << number << "! = " << factorialRecursive(number) << endl;
    }

    return 0;
}
  3. Save the file.

  1. Open the terminal in your IDE and navigate to the directory containing factorial_calculator.cpp.
  2. Type the following command to compile your application: 
    g++ factorial_calculator.cpp -o factorial_calculator
  3. Type the following command to run your application: 
    ./factorial_calculator
  4. Enter a positive integer and see its factorial calculated both iteratively and recursively.

Congratulations! You've created and run a factorial calculator in C++. Now you're ready to tackle more complex projects.
